# ロジックの組み立て（ピラミッドストラクチャー）

## ピラミッドストラクチャーとは

記事の主張を論理的に伝えるための構造。

```
総論（記事全体の主張・ゴール）
    ↓
主張① ② ③（総論を裏付ける理由・3〜4個）
    ↓
理由（なぜその主張が言えるか）
    ↓
事実・根拠（人によって変わらない情報）
    ↓
So What（だから読者はどう行動すべきか）
```

**各主張のSo Whatを合体させると総論になる**

---

## 各要素の定義

| 要素 | 意味 | 例 |
|------|------|-----|
| **総論** | 記事全体で言いたいこと・ゴール | 「リベルサスはオンライン診療での処方が最も安心でコスパが良い」 |
| **主張** | 総論を裏付ける理由（3〜4個） | 「個人輸入はリスクが高い」「オンライン診療なら正規ルートで安全」 |
| **理由** | 主張がなぜ正しいか | 「正規ルートでないため偽物の可能性がある」 |
| **事実・根拠** | 主張を支える普遍的な情報 | 「厚労省が個人輸入薬に注意喚起（公式資料）」 |
| **So What** | だから読者はどうすべきか | 「安さだけで選ぶと健康被害につながる危険がある」 |

---

## 主張の数について

- **3個以上**：人間は3個以上の根拠で納得して行動を起こせる
- **5個以上**：迷いが出やすいのでNG。整理して3〜4個に絞る

---

## ピラミッドストラクチャーの例

### ノウハウ記事の例：「AGAとは」

**総論（結論）**
AGAは遺伝やホルモン影響で進行する脱毛症であり、放置すると進行するため、早期に正しく理解して対処することが重要。

**主張① AGAは「男性ホルモン」と「遺伝」が大きく関与する**
- 理由：特定ホルモン（DHT）と遺伝的要因で発症しやすい
- 根拠：日本皮膚科学会の診療ガイドライン
- So What：生活習慣だけでなく体質として理解する必要がある

**主張② 進行パターンは特徴的で、放置すると悪化する**
- 理由：前頭部・頭頂部から薄くなる典型パターンを取る
- 根拠：「ハミルトン・ノーウッド分類」が広く利用されている
- So What：早期に気づくことで治療選択肢が広がる

**主張③ 治療法はいくつか存在する**
- 理由：薬物治療・生活改善・発毛医療など多様な選択肢がある
- 根拠：厚労省承認成分や海外臨床データ
- So What：自分に合った方法を選ぶことで進行抑制が可能

**結論（再提示）**
AGAは進行性脱毛症。早期に正しく理解し、信頼できる治療法を選ぶことが大切。

### CV記事の例：「リベルサス 通販」

**総論（結論）**
リベルサスは通販でも購入できるが、安全性と価格の観点からオンライン診療での処方が最も安心でコスパが良い。

**主張① 個人輸入の通販はリスクが高い**
- 理由：正規ルートでないため偽物・品質不明品の混在可能性がある
- 根拠：厚労省の注意喚起・偽造GLP-1薬の摘発事例
- So What：安さだけで選ぶと健康被害につながる危険がある

**主張② オンライン診療なら正規ルートで安全に入手できる**
- 理由：医師診察経由で国内薬局から正規品が配送される
- 根拠：各オンラインクリニックが薬機法準拠を明記
- So What：安全性重視なら通販ではなくオンライン診療を選ぶべき

**主張③ オンライン診療の方が価格メリットもある**
- 理由：まとめ買いや初回割引など、個人輸入より安いプランが多い
- 根拠：各クリニックの月額料金・割引キャンペーン
- So What：コスト面でも「オンライン診療」が有利

## 骨子への落とし方

ピラミッドが完成したら骨子に変換する：

| ピラミッド要素 | 骨子への変換 |
|--------------|------------|
| 総論 | まとめH2の内容 ＋ 記事ゴール |
| 主張 | H2見出し ＋ H2直下の導入文 |
| 理由・根拠 | H3の本文 |
| So What | H3末尾の締め ＋ CTA配置のタイミング |

---

## 総論・主張・結論の確認ポイント

- 各主張のSo Whatを合体したら総論になっているか
- 主張がMECE（モレなくダブりなく）になっているか
- 主張が3〜4個に収まっているか
- 結論がユーザーの「次の行動」まで具体化されているか
